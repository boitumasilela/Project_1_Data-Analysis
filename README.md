# SHEQ DATA ANALYSIS

## Project description

This analysis aims to visualize the SHEQ data and summarizes it for top management in order to identify gaps and solve high impacting variables.


## Data sources

The data a pulled from a factory where the data is tracked on an excel SHEQ data.csv file. 

## Tools

Excel(Data Cleaning)
SQL (Database analysis)
Power BI(Visualizations and summary)
